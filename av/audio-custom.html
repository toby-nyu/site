How It Works

This custom audio player demonstrates JavaScript control of HTML5 audio:

  • Play/Pause Button: Toggles between play and pause states using audio.play() and audio.pause()
  • Stop Button: Pauses playback and resets to beginning using audio.currentTime = 0
  • Time Display: Shows current playback time and total duration in MM:SS format, updated via timeupdate event
  • Seek Bar: Allows jumping to any position in the track by setting audio.currentTime
  • Volume Control: Adjusts audio volume from 0-100% using audio.volume property (0.0 to 1.0)

Key JavaScript Methods

  • audio.play() - Start playback
  • audio.pause() - Pause playback
  • audio.currentTime - Get/set playback position in seconds
  • audio.duration - Get total length in seconds
  • audio.volume - Get/set volume (0.0 to 1.0)
  • audio.paused - Check if audio is paused (boolean)
