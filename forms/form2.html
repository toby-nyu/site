Server-Side PHP Code (form2.php)

How This Server-Side Script Works

This PHP script is hosted on an external server at https://tobibrodie.co.uk/nyu/form2.php. When you submit the form-demo.html or form.html forms, the data is sent to this script, which processes and displays it back to the user.

Key Concepts:

  • POST Method: Form data is sent securely using HTTP POST
  • Data Sanitization: htmlspecialchars() prevents Cross-Site Scripting (XSS) attacks
  • Conditional Display: Only shows fields that were actually submitted
  • Security: Passwords are masked and never displayed in plain text
  • Responsive Design: Uses inline CSS to style the success page

Complete PHP Code

<?php
// Enable error reporting for debugging (remove in production)
error_reporting(E_ALL);
ini_set('display_errors', 1);

// Check if form was submitted
$formSubmitted = $_SERVER['REQUEST_METHOD'] === 'POST';

Key PHP Concepts Demonstrated

  • $_SERVER['REQUEST_METHOD']: Checks if the form was submitted via POST
  • $_POST superglobal: Accesses form data sent from the client
  • Null coalescing operator (??): Provides default value if field is missing
  • htmlspecialchars(): Converts special characters to HTML entities to prevent XSS
  • isset(): Checks if checkbox was selected
  • Conditional rendering: PHP if statements control what HTML is displayed
  • ucfirst(): Capitalizes first letter of strings for display
  • Inline CSS: All styles contained in the PHP file for portability
