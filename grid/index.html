About These Grid Pages

Three focused examples illustrate different CSS Grid concepts: explicit placement, responsive sizing, and full page layout using grid-template-areas. Use the navigation above to explore each implementation.

Grid Demo 1

Demonstrates explicit row / column placement, spanning, and overlay stacking via z-index. The footer is intentionally positioned outside the grid.

  • Manual grid-row-start/grid-column-end placement
  • Item spanning with grid-area shorthand
  • Overlay item using z-index
  • External footer positioning vs grid items

Responsive Grid

Shows how column counts and implicit rows adapt as viewport width changes, using grid-auto-rows to normalize heights.

  • Breakpoint-driven column adjustments
  • Implicit row sizing with grid-auto-rows
  • Progressive enhancement of layout density

Simple Page Grid

Applies grid to a page skeleton (header, main, aside, footer) combining fractional columns and fixed-height rows for a balanced composition.

  • Semantic regions mapped to grid cells
  • Fractional column sizing 3fr / 1fr
  • Viewport-based row heights (vh units)
  • Separation of primary and ancillary content
