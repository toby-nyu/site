Responsive Grid Example

This grid layout adjusts based on the screen size.

Notice how I have coded the grid to expect 4 rows - this works poorly size wise as the grid takes up space that isn't really needed when the page width is large.

It does seems to work well when the screen is around 450px wide as the grid is full in a 4 x 4 layout. But what happens if the screen size is even smaller?

The grid changes to 3 columns and 5 rows, but the row height for the row not explicitly declared is not the same.

To fix this second issue, try adding the following:

			grid-auto-rows: 100px;

This will ensure that any rows not explicitly declared in the grid-template-rows property will be 100px high.

Now this has been added, we can easily fix the first issue, as we no longer need to explicitly declare the number of rows, or their size, as the above will do this the same for all rows. This means we can remove the following declaration:

			grid-template-rows: 100px;
