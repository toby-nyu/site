About This Section

This section demonstrates essential JavaScript concepts including event handling, DOM manipulation, event propagation, and dynamic content management across multiple pages.

JavaScript Events Demo

This page provides interactive demonstrations of various JavaScript event types, including mouse, keyboard, form, and touch events.

Techniques Used:

  • Click Events: Button click handling with addEventListener('click')
  • Keyboard Events: Keydown event detection for Enter key using addEventListener('keydown')
  • Form Events: Form submission handling with preventDefault() to control default behavior
  • Mouse Events: Mouse move tracking over canvas element with coordinate logging
  • Touch Events: Touch start detection for mobile device interaction
  • Pointer Events: Drag functionality using pointer down events for universal input support
  • Inline Styles: CSS-in-HTML styling approach for self-contained demonstration
  • Console Logging: Event data output to browser console for debugging and learning

Multi-Page JavaScript Site

Techniques Used:

Event Bubbling and Propagation

This page provides an in-depth demonstration of event propagation phases (capturing, target, bubbling) and how to control event flow using stopPropagation().

Techniques Used:

  • Event Bubbling: Demonstrates default event bubbling behavior where events propagate from child to parent elements
  • Event Propagation Control: Uses stopPropagation() to prevent event bubbling when needed
  • Three-Phase Event Flow: Explains capturing phase, target phase, and bubbling phase with practical examples
  • Event Delegation: Demonstrates pattern of adding single listener to parent for handling multiple child events
  • Capture Phase Listeners: Shows { capture: true } option for listening during capture phase
  • Event Properties: Uses e.target and e.currentTarget to understand event flow
  • Nested Elements: Parent-child div structure to visualize event propagation
  • Visual Feedback: Interactive buttons and colored divs to show which elements respond to events
  • Educational Documentation: Extensive inline explanations with code examples and best practices

Key Learning Outcomes

  • Understand different JavaScript event types (click, keydown, submit, mousemove, touch, pointer)
  • Use addEventListener() to attach event handlers to DOM elements
  • Prevent default browser behaviors with preventDefault()
  • Implement event delegation patterns for efficient event handling
  • Control event propagation using stopPropagation() and stopImmediatePropagation()
  • Understand the three phases of event flow: capture → target → bubble
  • Separate concerns using external CSS and JavaScript files
  • Use defer attribute for proper script loading timing
  • Debug events using console logging and event properties
  • Structure multi-page applications with consistent navigation and layout
  • Apply inline vs. external styling approaches appropriately
  • Implement responsive navigation with flexbox and media queries
